# mcr_gig_guide

A site to you show you the latest live shows happening in Manchester. Find out information on each gig including, location, price and dates.
You can also sign up and add your own gigs to help promote them.

Made using -

MongoDB 3.2.2<br>
Express 4.16.4<br>
Node 10.15.3<br>
Bootstrap 4.3.1<br>
